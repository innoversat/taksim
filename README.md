## Taksim

Taksim is a tool for setting up Nginx with SSL certificates with ease.

## Prerequisites

- Python 3.10+ (of course)
- Nginx
- Certbot

## Installation

```bash
git clone https://github.com/innoversat/taksim.git
cd taksim
```

## Usage

```bash
sudo python main.py -d example.com -d www.example.com -p 3000
```
